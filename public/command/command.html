⚡ Backup Commands

🗄️ Database Backup

$ php artisan backup:database
📀 Starting database backup: domain_db_2026-03-10_14-30-00.sql
✅ Backup created: domain_db_2026-03-10_14-30-00.sql (245 MB)

📦 Full Backup

$ php artisan backup:full
📦 Starting full backup: domain_full_2026-03-10_14-30-00.zip
✅ Backup created: domain_full_2026-03-10_14-30-00.zip (125 MB)

🔄 List Backups

$ ls -la storage/backup/database/
-rw-r--r-- 1 user user 245M Mar 10 14:30 domain_db_2026-03-10_14-30-00.sql

🧹 Clean Old Backups

$ php artisan backup:cleanup
🧹 Cleaning up old backups...
✅ Removed 3 old backups

$ php artisan backup:status
